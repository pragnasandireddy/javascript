JAVASCRIPT EVENTS

Mouse Over Event

Change the color of the background from red to green on mouse over event

Mouse Out Event

Change the color of the background from blue to yellow on mouse out event

On Click Event

Change the color of the background from aqua to grey on click event

On Double Click Event

Change the color of the background from yellowgreen to orange on double click event

On Focus Event

Change the color of the background from beige to grey on focus event

On Blur Event

Change input from lower case to upper case on blur event
